
Перемещение по выделению build 12
Плагин для менеджера FAR 1.70
------------------------
Selection jump build 12
FAR 1.70 plugin


Позволяет перемещаться по выделенным файлам в активной
панели. Для удобства вешается макрос на клавиши, например
Alt+Home/Up/Down/End - соотв-но к
первому/предыд/следующ/последнему файлу.
-----------------------------------------------------------
Allows to move through the selected files in the active
panel. For best use, setup keymacroses for your favorite keys,
f.e. Alt+Home/Up/Down/End -> To first/prev/next/last file.
