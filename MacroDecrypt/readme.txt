  MacroDecrypt - приложение к ФАР Менеджеру, позволяет конвертировать
макросы в следующие формы строк:

1) из однострочной (REG_SZ)   --->  во внутреннюю многострочную (удобна для написания макроса);

только параметр - "Sequence":
2) из многострочной (REG_MULTI_SZ) ---> во внутреннюю многострочную;


3) из внутренней многострочной  ---> в однострочный макрос (REG_SZ);

только параметр - "Sequence":
4) из внутренней многострочной  ---> в многострочный макрос (REG_MULTI_SZ);



Примеры конвертирования reg-файлов MacroDecrypt'ом прилагаются...


 версия 2.3 (03.01.2008)
------------------------
[*] Старт сборки в одну строку начинается теперь с такой последовательности - "=" ранее было с - "Sequence".

    !!! ВНИМАНИЕ !!!
    последовательность - "=" должна быть на одной строке
    так правильно:
    [HKEY_CURRENT_USER\Software\Far\KeyMacros\Shell\CtrlDown]
    "Sequence"=" .....

    а вот так - НЕТ:
    [HKEY_CURRENT_USER\Software\Far\KeyMacros\Shell\CtrlDown]
    "Sequence"=
    "  .....
